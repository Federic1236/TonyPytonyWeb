The TonyPytony Language

A lightweight, Python-powered scripting engine with custom semantics and integrated Windows optimization.

Quick Installation

Install via setuptools and initialize your system environment with two simple commands:

pip install .
tony-setup

Core Syntax

TonyPytony Action Example
urla Print to Console urla "Ciao"
bomboroide Variable Assignment bomboroide x = 10
movimento Concatenation / Addition a movimento b
